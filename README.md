# Buzón de Sugerencias CBTIS 148

Aplicación desarrollada en Java utilizando NetBeans, diseñada para recopilar, gestionar y dar seguimiento a sugerencias, comentarios y propuestas de mejora de la comunidad educativa del CBTIS 148.

## Características

- Registro de sugerencias y comentarios.
- Gestión organizada de las propuestas recibidas.
- Interfaz gráfica intuitiva.
- Almacenamiento de información para su posterior consulta.
- Facilita la comunicación entre estudiantes y la institución.

## Tecnologías utilizadas

- Java
- NetBeans IDE
- Swing (Interfaz gráfica)
- MySQL (si utilizaste base de datos)
